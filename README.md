Описание проекта

Этот веб-сайт разработан для пекарни и предлагает удобный интерфейс для ознакомления с ассортиментом продукции, информацией о компании и контактными данными. Сайт выполнен с учетом современных стандартов и адаптирован для комфортного просмотра на любых устройствах — от настольных компьютеров до мобильных телефонов.

# Live: https://dmitriyverst.github.io/bakery/

Используемые технологии

HTML — для создания структуры страниц.
CSS — для оформления и стилизации элементов.
JavaScript — для реализации интерактивных функций и улучшения пользовательского опыта.
Ключевые особенности

Гибкий дизайн, который автоматически подстраивается под размер экрана.
Простая и понятная навигация, позволяющая быстро находить нужную информацию.
Чистая и модульная архитектура кода, что облегчает дальнейшее развитие и поддержку проекта.
