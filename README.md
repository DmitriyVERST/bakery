# Веб-сайт пекарни

Этот сайт предназначен для пекарни и предоставляет удобный интерфейс для просмотра ассортимента продукции, получения информации о компании и связи с клиентами. Он разработан с учетом современных стандартов веб-дизайна и адаптирован для комфортного использования на различных устройствах — от компьютеров до мобильных телефонов.

## Используемые технологии

- **HTML** — структура страниц и контента  
- **CSS** — оформление и стилизация элементов  
- **JavaScript** — реализация интерактивных функций и улучшение пользовательского опыта  

## Основные особенности

- Адаптивный дизайн, автоматически подстраивающийся под размер экрана  
- Простая и интуитивно понятная навигация для быстрого поиска информации  
- Чистая, модульная архитектура кода, что облегчает поддержку и развитие сайта
